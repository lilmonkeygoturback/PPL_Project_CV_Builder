🚀 Quick Start Guide

Upload your CV data in JSON format, and our system will automatically format it into a professional resume. The JSON should contain sections like personal_info, experience, education, skills, and more. If you need help or do not know how to use it, please click the "How it Works" button to get the instructions.

📋 Step-by-Step Instructions:
1
📄 Prepare Your JSON File

Create a JSON file with your CV data following this structure:

{
  "cv": {
    "personal_info": {
      "name": "Your Name",
      "email": "your.email@example.com",
      "phone": "+1234567890",
      "address": "Your City, Country",
      "linkedin": "linkedin.com/in/yourprofile",
      "website": "yourwebsite.com"
    },
    "summary": "Your professional summary...",
    "experience": [...],
    "education": [...],
    "skills": [...],
    "certifications": [...],
    "languages": [...]
  }
}
2
📤 Upload Your File

Drag and drop your JSON file into the upload area above, or click "Choose File" to browse and select it.

3
✨ Generate Resume

Click "Generate Beautiful Resume" and watch as your data transforms into a professional resume layout with stunning animations.

4
💾 Download & Share

Download your resume as PDF for job applications or PNG for social media sharing in ultra-high quality.

💡 Pro Tip:

Make sure your JSON file is properly formatted and includes all required fields for the best results. You can validate your JSON using online tools before uploading for optimal performance!
